# 📝 Blog Application (ContentNest)

A full-stack blogging platform built with React and Appwrite, allowing users to create, manage, and explore content with authentication, real-time updates, and content moderation.

🔗 **Live Demo:** https://contentnest.netlify.app

---

## 🚀 Features

- 🔐 Authentication (Login / Logout)
- 🧭 Navigation using React Router DOM
- ✍️ Create, edit, and delete blog posts
- 🖼 Upload blog posts with images
- 👁 View count system (1 view per user per hour)
- 🕒 Displays last updated time for posts
- 🔒 Public & Private blog visibility
- 🔍 Search blogs by username or title
- 🚫 Content moderation to prevent vulgar/inappropriate content
- 📈 "Most Viewed Today" section
- ⚡ Smooth animations using Framer Motion
- 💀 Skeleton loading screens for better UX

---

## 🛠 Tech Stack

- **Frontend:** React, Tailwind CSS, HTML, JavaScript  
- **Backend (BaaS):** Appwrite  

---

## 📦 Libraries & Tools

- **React Router DOM** – Navigation and routing  
- **React Hook Form** – Form handling and validation  
- **Framer Motion** – Animations and transitions  
- **HTML Parser** – Rendering dynamic content safely
